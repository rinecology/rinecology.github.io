Features:

  • ✅ Parse Ontario SPCOMP format (flexible and strict parsing)
  • ✅ YAML-based rule sets loaded at startup
  • ✅ Upload custom YAML rule files at runtime
  • ✅ Cross-rule-set comparison with diverging bar chart
  • ✅ Area column auto-detection (m², ha, record count)
  • ✅ Comprehensive HTML reports with charts
  • ✅ CSV export functionality
  • ✅ 100% browser-based — no server required
  • ✅ All data processing happens locally (privacy guaranteed)
